Objetivos del OVI

Objetivo general:
Apropiar los conceptos del curso diseño de sitios web y aplicarlos mediante el desarrollo de las estructuras básicas de HTML5 (etiquetas, atributos y valores).
✓ Objetivo específico 1:
Aprender a utilizar las diferentes etiquetas dentro de HTML5 entendiendo sus usos y funciones.
✓ Objetivo específico 2:
Entender cómo aplicar atributos a las diferentes etiquetas para la creación de un archivo HTML5.
✓ Objetivo específico 3:
Reconocer y aplicar valores a los diferentes elementos en la creación de un archivo HTML5.

objetivos del curso

Desarrollar sitios web en formato HTML5 y CSS3,
para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad,
por medio del uso de diferentes editores de código propuestos en el desarrollo del curso.

Tematicas del curso

Introducción a HTML5, Estructura básica de HTML5,
Aplicación de elementos de HTML5 a un proyecto de diseño Web, Introducción a CSS3, Estructura básica de CSS3,
Aplicación de elementos de CSS3 a un proyecto de diseño Web.
